PROJECT 001

Design MU Clean & Clear App

App Design UI/UX Figma

ออกแบบแอปพลิเคชันรายงานปัญหาภายในมหาวิทยาลัยมหิดล ด้วย Figma ครบถ้วนจากการวิจัยผู้ใช้ ไปจนถึง High-fidelity Design System

Scroll to explore

Overview

โปรเจกต์นี้เป็นการออกแบบแอปพลิเคชัน MU Clean & Clear สำหรับรายงานปัญหาภายในมหาวิทยาลัยมหิดล ด้วย Figma โดยออกแบบครบตั้งแต่ Low-fidelity Wireframe ไปจนถึง High-fidelity Design System พร้อม Interactive Prototype

Challenge

  • ออกแบบ User Interface ที่เข้าใจง่ายสำหรับการรายงานปัญหา
  • สร้าง Design System ที่สอดคล้องและสามารถนำไปใช้งานได้
  • ออกแบบ Dashboard สำหรับผู้บริหารในการติดตามปัญหา

Solution

ออกแบบแอปพลิเคชันด้วย 3 ส่วนหลัก:

User Interface

ออกแบบ UI ที่เรียบง่าย สวยงาม และใช้งานง่ายสำหรับทุกเพศทุกวัย

Design System

สร้าง Component Library และ Design Guidelines สำหรับการพัฒนา

Admin Dashboard

ออกแบบ Dashboard สำหรับผู้บริหารจัดการและติดตามปัญหา

Design Process

1. User Research

ให้ผู้ใช้กรอกแบบสอบถาม 40 คน เพื่อเข้าใจความต้องการและปัญหาในการรายงาน

2. Wireframing

สร้าง High-fidelity Wireframe และ User Flow แผนผังด้วย Figma

3. Visual Design

ออกแบบ Color Palette, Typography และ Component Library ด้วย Figma

4. Interactive Prototype

สร้าง High-fidelity Prototype และ User Testing เพื่อการปรับปรุง

Tools Used

Figma User Research Wireframe Prototype Design System UI/UX Design
