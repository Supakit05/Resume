Overview

MU Clean & Clear เป็นแอปพลิเคชันสำหรับรายงานปัญหาต่างๆ ภายในมหาวิทยาลัยมหิดล เช่น สิ่งแวดล้อม สิ่งอำนวยความสะดวก เพื่อให้ผู้บริหารสามารถติดตามและแก้ไขปัญหาได้อย่างรวดเร็ว

Development Scope

  • พัฒนา Frontend ด้วย React Native เพื่อให้ใช้งานได้ทั้ง iOS และ Android
  • สร้างระบบติดตามสถานะการแก้ไขปัญหา Real-time
  • ตั้งค่า Backend ด้วย MySQL
  • ทำ Unit Testing และ Integration Testing

Technical Implementation

เลือกเทคโนโลยีที่เหมาะสมสำหรับการพัฒนา:

Frontend Stack

  • Java (Android Native)
  • Dart (Flutter)
  • Material Design

Backend Stack

  • MySQL Database
  • Node.js

Key Features

Problem Reporting

ระบบรายงานปัญหาพร้อมการแนบรูปภาพ อธิบายรายละเอียด และเลือกหมวดหมู่ปัญหา

User Authentication

ระบบลงทะเบียนและเข้าสู่ระบบด้วย UserID/Password หรือ Mahidol Account

Status Tracking

ติดตามสถานะการแก้ไขปัญหา Real-time พร้อมการแจ้งเตือนเมื่อมีการอัปเดต

Development Process

1. Environment Setup

ตั้งค่า React Native project ด้วย Expo และ Firebase Console

2. Component Development

พัฒนา UI Components ตามการออกแบบ Figma และสร้าง Navigation Structure

3. Feature Integration

บูรณาการ Firebase Services, Image Upload และ Real-time Notifications

4. Testing & Deployment

ทำ Unit Testing, Integration Testing และ deploy ไป Google Play Store
