โปรเจกต์นี้เป็นการออกแบบและพัฒนาเว็บไซต์ e-commerce โดยมุ่งเน้นประสบการณ์ผู้ใช้งาน (UX/UI) ที่ใช้งานง่ายและทันสมัย พร้อมทั้งพัฒนาฟังก์ชันการซื้อขายออนไลน์ให้มีประสิทธิภาพ รองรับการใช้งานทั้งบนคอมพิวเตอร์และอุปกรณ์พกพา

Overview

โปรเจกต์นี้เป็นการออกแบบและพัฒนาเว็บไซต์ e-commerce โดยใช้เครื่องมือ Figma สำหรับการออกแบบ UI/UX และพัฒนาด้วย HTML, CSS และ JavaScript รวมถึงมีการจัดเก็บและบริหารจัดการข้อมูลด้วย MySQL เพื่อรองรับการทำงานของระบบอย่างมีประสิทธิภาพ

Objective

  • ออกแบบ User Interface ที่เป็นมิตรและใช้งานง่ายด้วย Figma
  • พัฒนาระบบเว็บไซต์ที่รองรับการขายสินค้าออนไลน์
  • สร้างระบบจัดการสินค้า ผู้ใช้งาน และการชำระเงิน

Project Scope

Frontend Design

ออกแบบหน้า UI/UX และพัฒนาด้วย HTML, CSS, JavaScript

Backend Development

สร้างระบบฐานข้อมูล และ API สำหรับการจัดการข้อมูล

Technologies & Tools

Design

Figma

Frontend

HTML, CSS, JavaScript

Database

MySQL

Key Features

Product Catalog

แสดงสินค้าแบบหมวดหมู่ พร้อมรายละเอียดและราคา

Shopping Cart

ระบบเลือกสินค้าและจัดการรถเข็น

Payment Gateway

รองรับการชำระเงินหลายช่องทาง

User Management

ระบบสมัครสมาชิก เข้าสู่ระบบ และจัดการโปรไฟล์

Tools Used

Figma HTML CSS JavaScript MySQL E-commerce
